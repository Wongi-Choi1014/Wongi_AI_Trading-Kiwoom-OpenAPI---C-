# Wongi_AI_Trading_Kiwoom_OpenAPI_C#
본 프로그램은 키움(Kiwoom) OpenAPI를 활용하여 AI 및 알고리즘 매매를 수행한다. AI에 강력한 Python기반으로 제작하려 하였으나, API사용이 용이하고 GUI 기반에서 편리한 코드 작성이 가능한 C#을 최종 선택하였다. <br>
